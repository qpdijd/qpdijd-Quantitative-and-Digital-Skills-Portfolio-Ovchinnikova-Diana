 Project 2 – Customer Base Audit
This project evaluates the customer base of an online retailer using transactional data.

Skills Developed:

Conducted a customer base audit using transactional data to evaluate customer heterogeneity, cohort dynamics, and revenue concentration.
Applied quantitative techniques including decile analysis, cohort analysis, and customer segmentation to identify differences in purchasing behavior across customer groups.
Analyzed key performance metrics such as average spending per customer, purchase frequency, and revenue distribution. 
Created analytical visualizations and interpreted statistical patterns to assess customer retention, customer value evolution, and overall business health.
Demonstrated data analysis and business analytics skills using spreadsheet tools to transform raw data into actionable managerial insights.
